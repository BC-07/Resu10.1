# ResuAI - University Personnel Assessment System

ResuAI is a comprehensive university personnel assessment and recruitment management system designed specifically for academic institutions. It combines advanced AI-powered document processing with sophisticated assessment workflows to streamline faculty and staff hiring processes.

ResuAI is designed for **university HR departments** and **academic recruitment teams** to:

- Track candidates through the hiring pipeline
- Generate detailed assessment reports and rankings

### Production Setup

#### Database Configuration

**For SQLite (Development):**

- Automatic setup - no configuration needed
- Database file: `resume_screening.db`
- Suitable for single-user or testing

**For PostgreSQL (Production):**

1. **Install PostgreSQL server**
2. **Create database:**

```sql
CREATE DATABASE resumai_university;
CREATE USER resumai_user WITH PASSWORD 'secure_password';
GRANT ALL PRIVILEGES ON DATABASE resumai_university TO resumai_user;
```

3. **Configure environment variables:**

```env
DATABASE_URL=postgresql://resumai_user:secure_password@localhost:5432/resumai_university
FLASK_ENV=production
SECRET_KEY=your-production-secret-key
```

#### Web Server Deployment

**Using Gunicorn (Recommended):**

```bash
pip install gunicorn
gunicorn -w 4 -b 0.0.0.0:5000 --timeout 300 app:app
```

**Using Docker:**

```dockerfile
FROM python:3.9-slim
COPY . /app
WORKDIR /app
RUN pip install -r requirements.txt
EXPOSE 5000
CMD ["gunicorn", "-w", "4", "-b", "0.0.0.0:5000", "app:app"]
```

## ğŸ“ Assessment Methodology

### University Scoring System

**Category Breakdown:**

1. **Education (40% weight)**
   - Relevance/Appropriateness to position
   - Basic minimum requirements met
   - Doctoral progress (if applicable)
   - University prestige and accreditation

2. **Experience (20% weight)**
   - Years of relevant professional experience
   - Leadership and management roles
   - Industry-specific experience
   - Academic experience (teaching, research)

3. **Training (10% weight)**
   - Professional development hours
   - Certifications and specialized training
   - Continuing education
   - Skills enhancement programs

4. **Eligibility (10% weight)**
   - Professional licenses
   - Civil service eligibility
   - Board examinations passed
   - Legal requirements for position

5. **Accomplishments (5% weight)**
   - Awards and recognitions
   - Publications and research
   - Community involvement
   - Professional memberships

6. **Potential (15% weight)**
   - Interview performance (manual entry)
   - Aptitude test scores (manual entry)
   - Leadership potential assessment
   - Cultural fit evaluation

### Assessment Workflow

**Automated Processing:**

1. **Document Upload** â†’ PDS/Resume parsing
2. **Data Extraction** â†’ Education, experience, training details
3. **Initial Scoring** â†’ Categories 1-5 automatically calculated
4. **Validation** â†’ Review extracted data for accuracy

**Manual Assessment:**

1. **Interview Scheduling** â†’ Candidate interview process
2. **Score Entry** â†’ Manual input of interview and aptitude scores
3. **Final Calculation** â†’ Combined automated + manual scores
4. **Ranking** â†’ Position ranking with recommendations

**Quality Assurance:**

- **Multiple Reviewers** â†’ Cross-validation of scores
- **Score Justification** â†’ Required notes for manual adjustments
- **Audit Trail** â†’ Complete history of score changes
- **Standardization** â†’ Consistent criteria across positions

## ğŸ” Troubleshooting Guide

### Common Issues

**Database Errors:**

```
Error: sqlite_assessment_db module not found
Solution: System uses direct SQLite connections - this error is expected
```

**File Processing Issues:**

```
Error: PDS extraction failed
Solution: Check file format, ensure it's a valid PDS document
```

**Assessment Calculation Errors:**

```
Error: Missing assessment scores
Solution: Verify all required data is extracted from PDS
```

```

Solution: Check default admin credentials or reset password
```

**For Large Datasets:**
